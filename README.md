Description:

Block Game is a Java-based game that consists of multiple classes. The main class to execute the game is BlockGame.



How to Run in IntelliJ:

1. Open IntelliJ IDEA.

2. Click on File > Open and select the folder containing the project files.

3. Ensure the JDK is properly configured under File > Project Structure > SDKs.

4. Click on Run > Edit Configurations.

5. Add a new Application configuration:

6. Set the Main class to BlockGame.

7. Ensure the working directory is set correctly.

8. Click Run or Debug to start the game.




Requirements:

IntelliJ IDEA installed

Java Development Kit (JDK) configured in IntelliJ



Notes:

Ensure all Java files are properly recognized by IntelliJ.

If encountering any issues, check the Run/Debug Configurations and ensure the correct main class is selected.
